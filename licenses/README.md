# licenses/

肖像/姓名/形象 **授权书** 的登记目录（扫描件建议加密存储，Git 中可只保留元数据）。

## 建议每条授权

- `license_id`：如 `LIC-2026-001`
- `subject_id`：对应 `known_faces` 主体
- `scope`：用途（广告、短视频、封面、全渠道等）
- `valid_from` / `valid_until`
- `territory`、`exclusivity`（如有）
- `evidence_path`：内部存储路径或案件引用（**不要**写公网可下载裸链）

可在本目录使用 `LIC-2026-001.yaml` 等文件；敏感 PDF 放在加密卷或对象存储，此处只记索引。
